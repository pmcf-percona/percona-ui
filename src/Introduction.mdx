# Welcome to Peak Design

Peak Design is Percona's design system and UI library containing the guidelines and assets that allow Perconians and the community to create functional, professional, and attractive GUIs for our software products.

While our Figma kit represents our design intent, **this Storybook is our definitive coded source of truth**. What you see here is exactly what renders in the UI codebase and Percona products.

## Resources

- [Source code](https://github.com/percona/percona-ui): The source code for the Peak Design library and Storybook.
- [MUI docs](https://mui.com/material-ui/): The core library that Peak Design is built on top of.
- [Material React Table docs](https://www.material-react-table.com/): A library for the advanced data tables we use.
- [Figma kit](https://www.figma.com/design/08jGF3GZAUGmazlQtk0UQk/Peak-Design-Kit): If you're interested in making mockups for new software ideas or iterate on Percona products.

## Recommended workflow

For the best experience, treat this Storybook as your first stop:

1. **Search Storybook first**: Identify if the component or token you need is already built and documented here.
2. **Verify the design**: Use the "Design" tab, if available, to ensure the coded component matches the Figma spec.
3. **Implement**: Copy the code snippets and use them in your feature branch.
4. **Missing a component?** If a component exists in the Peak Design Figma kit or MUI but isn't here yet, it's time to build it and add it to the Storybook! Contact us to learn about the contribution guidelines.
