# Typography

Typography in Peak Design is based on [MUI Typography](https://mui.com/material-ui/react-typography/) with the Percona branding applied.

## Headings

These styles establish a clear visual hierarchy and set the tone across the product. Each level, ranging from `h1` to `h6`, has distinct sizes and weights for a consistent structure.

<Canvas of={TypographyStories.HeadingsShowcase} sourceState="none" />

## Titling

Complementary styles for titling and subheadings used to support document composition using the heading styles.

<Canvas of={TypographyStories.TitlingShowcase} sourceState="none" />

## Body styles

These styles are used for the main body text of the product. They are designed to be easy to read and follow, and are used for paragraphs, lists, and other block-level content.

<Canvas of={TypographyStories.BodyShowcase} sourceState="none" />

## Component text styles

Text styles used within interactive components like buttons, menus, and form elements. They are stored in the library to lock the style and extend the consistency across components. More styles may be added in the future as needed in GUI development.

<Canvas of={TypographyStories.ComponentStylesShowcase} sourceState="none" />

## Code

The `code` style is the centralized monospace identity (font-family, weight, letter-spacing) shared by all code surfaces. Size and line-height are left to each consumer, since they are context-dependent and mostly used inline. For code snippets, use the `CodeBlock` component.
