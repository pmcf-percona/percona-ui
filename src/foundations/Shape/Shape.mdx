# Shape

Shape tokens define the border radius scale for Peak Design. They give components a single, consistent source for corner rounding, replacing MUI's default `borderRadius` and any hard-coded pixel values scattered across the codebase.

The scale is a curated set of steps (`3, 5, 8, 13, 21`, plus fully rounded). `0px` is the implicit default — sharp corners need no token. `borderRadiusSm` (`5px`) is the theme-wide default applied to any component that doesn't set its own radius.

## Tokens

Every token lives on the theme at `theme.shape.*` and is shared across all themes (`base`, `pmm`, `sep`).

## Fully rounded

`borderRadiusFull` (`999px`) rounds corners completely: a circle on square elements, a pill on wider ones.

## Usage

Always draw radius from the theme so components stay in sync as the scale evolves — never hard-code pixel values.

```tsx
import { useTheme } from '@mui/material';

const theme = useTheme();

// Read a specific token
theme.shape.borderRadiusMd; // 8

// Or in an sx prop
<Box sx={{ borderRadius: `${theme.shape.borderRadiusLg}px` }} />;
```

Components that don't set a radius inherit the theme default (`borderRadiusSm`, `5px`) automatically via MUI's `theme.shape.borderRadius`.
