# Primitives

These are the foundational color scales used to build semantic tokens. Primitives are not affected by light/dark mode. They remain constant across all themes.

Use primitives to construct semantic tokens in your theme (e.g., `error.main` might use `red[600]`).

## Brand colors

These swatches represent the Percona brand. Use them to feed your work as the foundation to build new UI palettes.

## Extra colors

For messaging we will need these extra swatches, so we can have "true red" and "true green", for example, for messaging error or success in the typical GUI paradigm. These palettes match the same structure as the brand swatches, so they act as a natural extension but more focused on the messaging intent.
